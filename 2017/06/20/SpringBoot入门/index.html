入门

SpringBoot是Spring升级版。

创建方式

第一种

创建Spring Initializr项目

选择Maven构建项目

添加Web依赖

创建项目后可以看到，在Pom.xml中新增了SpringBoot相关的依赖，并在src目录中新增application.properties和Application.class文件。

第二种

直接创建Maven工程，在Pom.xml中添加SpringBoot相关依赖，在项目中创建application.yml文件和Application相关类（添加@SpringBootApplication注解）。

使用

创建一个类，添加@RestController注解。

@RestController是Spring4新增注解，@Controller和@ResponseBody的结合。

访问localhost:8080即可看到结果。

定时任务

可以使用注解@Scheduled(corn="")执行定时任务，在类中添加@EnableScheduling开启定时任务。

一个corn表达式依次是：

  • 秒（0-59）
  • 分（0-59）
  • 时（0-23）
  • 天（0-30，需要考虑月份）
  • 月（0-11）
  • 天（1-7，星期几）

其中每个元素可以是：

  • 一个值（10）
  • 一个连续区间（1-2）
  • 一个间隔时间（1-10/2，／代表每隔两个小时）
  • 一个列表（1，2，3）

由于月份中某天和星期中某天互斥，其中一个必须设置为？,*代表通配符。
